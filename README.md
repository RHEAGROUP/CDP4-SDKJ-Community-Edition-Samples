The Concurrent Design Platform Software Development Kit is a Java SDK that is compliant with ECSS-E-TM-10-25A Annex A and Annex C. The following libraries are made available in the Community Edition under the [GNU LGPL](https://www.gnu.org/licenses/lgpl-3.0.en.html):

  - CDP4Common 
  - CDP4Dal
  - CDP4ServicesDal

This repository contains example projects that demonstrate how the SDK can be used.
The source code of the SDK can be found [here](https://github.com/RHEAGROUP/CDP4-SDKJ-Community-Edition).
